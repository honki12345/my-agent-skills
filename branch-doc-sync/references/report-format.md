# Branch Docs Update Summary Format

## Branch Diff Summary
- Base branch and compared range
- Changed file count and key areas

## Managed Documentation Inventory (Full Coverage)
- Table: doc path | decision (`update`/`no-change`) | reason | matched source files
- Requirement: every managed doc appears exactly once

## Missing Documentation Candidates
- Table: doc path | decision (`add`) | reason | matched source files

## Doc Impact Matrix
- Table: doc path | impact reason | source files

## Applied Doc Edits
- Table: doc path | section edited | change summary

## Validation Checklist
- [ ] All managed docs reviewed (no omissions)
- [ ] Commands/examples executed or syntax-checked
- [ ] Env vars and config keys verified
- [ ] Internal links and file paths verified

## Follow-up Notes
- Explicit TODOs or assumptions when evidence is incomplete
