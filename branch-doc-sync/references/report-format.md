# Branch Docs Update Summary Format

## Branch Diff Summary
- Base branch and compared range
- Changed file count and key areas

## Doc Impact Matrix
- Table: doc path | impact reason | source files

## Applied Doc Edits
- Table: doc path | section edited | change summary

## Validation Checklist
- [ ] Commands/examples executed or syntax-checked
- [ ] Env vars and config keys verified
- [ ] Internal links and file paths verified

## Follow-up Notes
- Explicit TODOs or assumptions when evidence is incomplete
